About the project

~

Description


"Clouds Rescue" is an educational 2D mobile game designed to make learning math fun and interactive for 10th-grade students. Through an engaging storyline and challenging levels, this game offers a unique opportunity to reinforce mathematical concepts while having an enjoyable gaming experience.


Features


Clouds Rescue comes with an interactive storyline that engages players in a rescue mission. It also includes challenging math questions and puzzles at each level, educational content aligned with the 10th-grade curriculum, and a variety of mathematical topics covered, including algebra, geometry, and calculus. The dynamic gameplay encourages critical thinking and problem-solving, and it provides progress tracking to monitor mathematical skill development. The game is enhanced with visually appealing 2D graphics and animations.


Educational Value


The primary goal of "Clouds Rescue" is to enhance understanding of mathematical concepts among students. It also aims to reinforce problem-solving and critical-thinking skills. Furthermore, the game provides a practical application of math concepts in a gaming context and prepares students for exams and tests by offering real-world math challenges. Ultimately, it aims to foster a love for learning by making math enjoyable and engaging.


Requirements


To enjoy "Clouds Rescue," you'll need Unity 2021.3.25f1 or a later version. Additionally, you'll need an Android device for installation.


Installation


To install the game, follow these simple steps:

1. Download the APK file from the Releases section.

2. Install the game on your Android device.
